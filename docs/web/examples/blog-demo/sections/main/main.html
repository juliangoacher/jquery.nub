Welcome the 'jquery.nub' blog, a demo UI application created with jquery.nub.

This example is fully running in the browser. Only JavaScript and HTML code has been used and all data is stored in the browser DOM using the jquery.nub plugin.

The blog is intended to show how to work with jquery.nub frames and dynamicMenu

  • The main menu is an example of a nubDynamicMenu.
  • In the admin section you can make post and there is also an dynamicFrame example driven with a list navigate subsections.

Click on the main section to make a new post.
